Listas Desordenadas

Las listas desordenadas utilizan viñetas para enumerar los elementos de la lista. Para iniciar una lista desordenada se utiliza el tag ul y li. Antes de cada uno de los elementos de la lista se usa el tag li.

Observa como se ve una lista desordenada simple:

Los libros recomendados para leer este mes son:

  • Los miserables
  • El Quijote de la Mancha

Es posible también añadir listas desordenadas dentro de otras listas desordenadas. Por ejemplo:

Libros interesantes

  • Los Miserables
    • Novela romántica francesa
  • El extranjero
    • Literatura contemporánea
    • El protagonista se siente extraño en su propio medio
  • El Quijote de la Mancha
    • El autor es Miguel de Cervantes Saavedra
    • La historia del ingenioso Hidalgo
