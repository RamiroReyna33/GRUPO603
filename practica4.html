En esta página conoceremos el uso de los principales tags de HTML. Lo primero que hay que observar es el esquema de colores del documento, incluido dentro del tag body. El esquema de colores se define con los atributos bgcolor, text, link y vlink. Puedes colocar solo los atributos que necesites dentro del tag body.

Analiza este párrafo para comprender el uso del tag de tabulación blockquote. Repetiremos varias veces este párrafo para comprender el uso del tag de tabulación blockquote.

La única diferencia entre un tipo y otro es el número o viñeta que identifica a cada elemento de la lista.
